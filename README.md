# 🏠 Homelab Infrastructure Projects

This repository contains hands-on configurations, documentation, and setup walkthroughs for a fully functional homelab infrastructure that mimics real-world enterprise environments.

## 📁 Project Structure

- `1_networking_basics` – Static/DHCP IPs, hostname, DNS settings
- `ansible_automation` – Automated tasks with Ansible
- `backup_disaster_recovery` – Backup strategies & recovery plans
- `centralized_logging_monitoring` – Central log collection and service monitoring
- `containerization_podman` – Podman container builds and deployment
- `linux_server_builds` – Linux install procedures and configurations
- `windows_ad_dns` – Windows AD domain and DNS setup

> This project is actively evolving as part of a full system engineering lab.
